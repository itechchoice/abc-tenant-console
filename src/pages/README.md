# pages 模块说明

`src/pages/` 存放路由页面及其页面私有组件。页面目录承担“组合模块”的职责，而不是沉淀所有复用逻辑。

## 当前页面

- `Home/`
  - 主工作台页面
  - 由左侧会话栏、中间聊天面板、右侧工作流画布构成
- `Login/`
  - 登录页
- `NotFound/`
  - 404 页面

## 组织原则

1. 页面只负责布局编排、跨模块协调和局部状态持有。
2. 页面私有组件放在对应页面目录下的 `components/`。
3. 只有跨页面复用的组件才提升到 `src/components/`。
